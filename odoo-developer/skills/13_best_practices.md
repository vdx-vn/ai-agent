# Skill: Odoo Best Practices

## Rules
- Keep modules small and focused
- Separate concerns: models vs wizard vs report
- Name XML ids consistently
- Avoid hardcoding company/currency assumptions
- Use i18n correctly

## Review checklist
- Security included?
- Multi-record safe?
- Upgrade-safe?
- No silent sudo abuse?
- Views robust xpaths?

## Example Prompt
"Review this PR for Odoo best practices and potential security issues."
